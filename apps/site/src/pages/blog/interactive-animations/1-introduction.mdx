# Introduction to Interactive Animations

In this series, we'll explore how to create engaging, interactive Matplotlib animations directly in your browser. Unlike traditional animation libraries like Manim, which require Python installation and render videos, our approach runs completely in the browser.

## How It Works

We're using [Pyodide](https://pyodide.org/), a Python distribution that runs in the browser via WebAssembly. This allows us to:

1. Run Python code directly in the browser
2. Use NumPy and Matplotlib for computations and visualization
3. Create interactive animations using HTML5 canvas

## Example: Linear Regression Animation

Here's a simple example showing how a linear regression model fits a line to data points:

<PyodideMatplotlib code={animationCode} fileName="linear_regression.py" height="400px" readOnly={true} />

## How the Animation Works

The animation above demonstrates:

1. Data points appearing one by one
2. The regression line being drawn
3. Error lines showing the distance between points and the line

This helps visualize how linear regression works - by minimizing the sum of squared errors between the predicted line and the actual data points.
