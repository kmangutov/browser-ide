# Create Your Own Animations

Now that we've seen a few examples, let's explore how you can create your own animations using our framework. The `AnimationHelper` class makes it easy to define complex animations that run efficiently in the browser.

## Getting Started

Creating a custom animation requires defining a class with two main methods:

1. `setup()` - Called once to initialize the figure, axes, and data
2. `update(frame)` - Called for each animation frame to update the visualization

## Benefits Compared to Traditional Manim

This approach has several advantages over traditional Manim for certain use cases:

- **No installation required**: Everything runs in the browser
- **Interactive**: Users can play, pause, and restart animations
- **Editable**: With the code editor, users can modify and experiment
- **Familiar API**: Uses standard Matplotlib, not a custom animation language

## Limitations

There are some limitations compared to Manim:

- Less sophisticated camera control and effects
- Fewer pre-built animation primitives
- Limited to Matplotlib's visualization capabilities
- Slower than pre-rendered animations
