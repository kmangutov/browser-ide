# Decision Boundaries Visualization

Decision trees split data recursively to classify samples. The visualization below shows how the decision boundaries evolve as the tree depth increases.

## Interactive Tree Visualization

The animation shows how a decision tree creates boundaries that divide the feature space:

<PyodideMatplotlib code={animationCode} fileName="decision_tree.py" height="400px" readOnly={true} />

<MarkdownWrapper>

## Understanding Tree Depth

Notice how increasing the tree depth:

1. Creates more complex decision boundaries
2. Better fits the training data
3. Can lead to overfitting when depth is too high

The rectangular regions correspond to the rules created by the decision tree. Each split in the tree creates a new boundary, which results in these perpendicular lines.

## How It Works

The animation uses scikit-learn's `DecisionTreeClassifier` to:

1. First show the data points for two classes
2. Then visualize the decision boundary with increasing tree depth
3. The boundary colors represent the predicted class for each region
